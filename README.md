# LGPD Rio Claro - Sistema de Conformidade

Este projeto é um sistema completo para gestão da LGPD (Lei Geral de Proteção de Dados) para a prefeitura de Rio Claro, RJ.

## 🚀 Como Rodar em Casa (Setup no Mac/Linux/Windows)

### Pré-requisitos
- **Node.js** (v18+)
- **Docker & Docker Compose** (Opcional, mas recomendado)
- **Git**

### Passo 1: Clonar o Repositório
```bash
git clone https://github.com/DalmoVieira/lgpd-rioclaro.git
cd lgpd-rioclaro
